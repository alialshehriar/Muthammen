# Vercel Deployment Instructions

This project is ready to deploy on Vercel.

## Quick Deploy

1. Go to https://vercel.com/new
2. Import from GitHub: alialshehriar/Muthammen
3. Framework: Vite
4. Build Command: `npm run build`
5. Output Directory: `dist`
6. Click Deploy

## Domain Setup

After deployment, add your domain:
- muthammen.com
- www.muthammen.com

Then configure DNS in Cloudflare with CNAME records pointing to Vercel.
